# Swiggy Sales Analysis Using Python

## Project Overview

This project analyzes Swiggy food delivery sales data using Python.

The analysis focuses on:

- Total Sales
- Average Rating
- Average Order Value
- Total Orders
- Monthly Sales Trends
- Daily Sales Trends
- Veg vs Non-Veg Revenue Analysis
- Revenue by State
- Quarterly Performance Summary
- Top 5 Cities by Sales

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Key KPIs

- Total Sales
- Average Rating
- Average Order Value
- Total Orders
- Rating Count

## Visualizations

- Monthly Revenue Trend
- Daily Revenue Trend
- Veg vs Non-Veg Revenue Contribution
- Revenue by State
- Quarterly Performance Summary
- Top 5 Cities by Sales
